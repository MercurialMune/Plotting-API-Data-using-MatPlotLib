Charting Ona API JSON Data using Django and MatPlotLib

How willing are Kenyans in accessing mental healthcare?


Notes on the graph beside

by Cyprian Munene

The following graph was generated from data generated by consuming the Ona Data API. The response was acquired from querying data from the penetration of community health professionals in Kenya. From the hits of the data, the sample represented a fair proportion of the entire country seeing as ethic groups and localities were distributed relatively proportionately. The data has been synthesized and cleaned using Python; The graph has been plotted and generated by integrating the matplotlib library.
